# 3TierProjectDeploy

This repository contains my 3-tier application deployment project. I deployed a 3-tier architecture on the AWS cloud platform, following the steps outlined in [this blog post](https://medium.com/@aaloktrivedi/building-a-3-tier-web-application-architecture-with-aws-eb5981613e30).

## Project Overview

- **Architecture:** 3-tier (Presentation, Application, Database)
- **Cloud Platform:** AWS

## Reference

- [Building a 3-Tier Web Application Architecture with AWS](https://medium.com/@aaloktrivedi/building-a-3-tier-web-application-architecture-with-aws-eb5981613e30)

## How to Use

1. Clone this repository.
2. Follow the instructions in the referenced blog to deploy your own 3-tier application on AWS.
